# Source-first React component library

Exre UI is initialized as a source-first React component library using Vite, pnpm, shadcn/ui with the Radix base, and preset `b5Kc86Jl4`. The library keeps the full shadcn/ui component source in `src/components/ui` so Exre can own and evolve its brand components while still distributing a package entry and stylesheet that consuming UI apps can depend on.
